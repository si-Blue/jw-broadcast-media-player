# JW Broadcast Media Player (Unofficial)

This is an unofficial webOS application for browsing and watching content from JW Broadcasting.

## Features

*   **Browse Content:** Navigate through the latest videos, video categories, and audio programs.
*   **Playback:** Watch videos and listen to audio with a built-in player.
*   **Search:** Find specific content with the search functionality.
*   **TV Optimized:** Designed for navigation with a TV remote (spatial navigation).
*   **Resume Playback:** The app saves your progress in videos.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have the webOS TV SDK installed and configured on your computer.

### Installing

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/jw-broadcast-media-player.git
    ```
2.  Install the dependencies:
    ```bash
    npm install
    ```
3.  Package the application:
    ```bash
    npm run package
    ```
4.  Install the application on your webOS TV or emulator:
    ```bash
    npm run install
    ```
5.  Launch the application:
    ```bash
    npm run launch
    ```

You can also use the combined script to build, install, and launch the application in one step:
```bash
npm run build-install-launch
```
